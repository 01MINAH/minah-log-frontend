# 비행기 컨셉의 UI 디자인

안녕하세요! "비행기"처럼 드넓은 하늘을 향해 끊임없이 도약하고 높이 날아오르는 개발자 민아의 포트폴리오 메인 콘셉트, **비행기 UI/UX 디자인 스토리**를 들려드립니다.

어떻게 하면 수많은 개발자 포트폴리오 속에서 나만의 색깔과 정체성을 확고하게 어필할 수 있을까? 라는 고민 끝에 도출된 비행기 메타포의 전반적인 레이아웃 설계 과정과 디자인 디테일에 대해 정리해 보았습니다.

---

## 1. 콘셉트의 시작: 도약과 도전의 날개

'비행기'는 저에게 여러 가지 상징적 의미를 담고 있습니다:
- **도전**: 중력을 이겨내고 하늘로 향하는 강인한 추진력
- **성장**: 더 높은 고도로 안착하기 위한 쉼 없는 비행
- **공유**: 먼 거리를 연결해 주는 비행기처럼, 지식과 가치를 나누는 소통의 다리

포트폴리오의 아이콘과 애니메이션 요소를 비행기 실루엣으로 통일하고, 전반적인 색감과 인터랙션을 가벼우면서도 정돈된 모션으로 구성하여 사이트 전체에 일체감을 주도록 구성했습니다.

---

## 2. 디자인 키 비주얼 (Visual Identity)

비행기 콘셉트를 구현하기 위해 다방면의 인터랙티브 요소를 가미했습니다:

### ① 로고 애니메이션 (Paper Airplane Motion)
헤더의 "MIN-AH LOG" 로고 왼쪽에 배치된 종이비행기 SVG 아이콘은 3D 공간을 선회하는 느낌을 주기 위해 CSS Keyframes 애니메이션을 사용해 무한히 위아래로 둥실거리며 비행하는 효과를 부여했습니다.

```css
@keyframes fly {
  0%, 100% { transform: translate(0, 0) rotate(-45deg); }
  50% { transform: translate(4px, -4px) rotate(-42deg); }
}

.logo-icon {
  animation: fly 4s infinite ease-in-out;
}
```

### ② 하늘을 담은 히어로 이미지
메인 홈 화면 진입 시 최상단에 푸른 하늘과 종이비행기가 날아가는 감성적인 배너 영역을 두어, 포트폴리오에 어울리는 감성적 몰입감을 형성합니다. 파스텔 베이지 톤과 푸른 하늘 에셋이 자연스럽게 오버랩되도록 미세한 세피아 필터를 추가했습니다.

---

## 3. 정돈된 레이아웃: 중앙 정렬의 편안함

포트폴리오 콘텐츠의 최우선 순위는 **'가독성'**입니다. 화려한 가로 레이아웃보다는 시선이 상단에서 하단으로 피로감 없이 자연스럽게 흘러내릴 수 있도록 **최대 860px 너비의 중앙 정렬 컨테이너**를 디자인 시스템으로 구축했습니다.

- **넓은 여백(Margin/Padding)**: 문단 사이와 섹션 간격을 충분히 띄워 텍스트가 빽빽하지 않고 눈이 편안해지도록 유도했습니다.
- **얇고 차분한 Border 라인**: 섹션이나 카드 영역을 굵은 그림자 대신 1px 굵기의 미세한 `--border-color` 테두리로 감싸, 정돈된 바인더나 노트와 같은 깔끔한 완성도를 선사합니다.

---

## 4. 파스텔 베이지와 하늘빛의 조화

비행기라는 주 콘셉트는 자칫 푸른색 계열만 과다 사용하여 너무 차갑고 흔한 IT 포트폴리오 느낌을 줄 수 있습니다. 이를 보완하기 위해 **파스텔 웜 베이지(`#FAF8F5`)**를 메인 베이스 색상으로 정하고, 옅은 갈색(Taupe)과 샌드 그레이를 포인트로 조합했습니다.

- **기본 배경**: 포근한 미색 한지나 에코 종이 느낌의 웜 화이트 베이지
- **포인트 블루/그린**: 하늘빛의 청량함을 담은 포인트들을 아이콘 뱃지나 링크 호버 시 아주 옅고 세련되게만 차용

이로써 기분 좋은 아날로그 감성과 모던한 웹 기술이 유연하게 공존하는 나만의 독특한 포트폴리오 아이덴티티가 완성되었습니다.
