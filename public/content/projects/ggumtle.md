# 꿈틀 (Ggumtle)

우리 아이의 꿈의 틀을 잡아주는 **맞춤형 아동 성향 진단 및 도서 추천 서비스**입니다.

## 프로젝트 소개
아동 심리 및 성향 분석 검사를 간단한 문답 형식의 웹 기반 진단 테스트로 제공하고, 분석 결과에 부합하는 도서들을 지능형 카테고리 태그 시스템을 기반으로 자동 추천하여 자녀 양육과 독서 습관 형성을 돕는 유익한 플랫폼입니다.

## 주요 특징 및 기능
- **간단 아동 성향 진단**: 귀여운 일러스트 캐릭터 문답을 통한 자녀 성향 유형화(MBTI 아동 판).
- **자동 맞춤 독서 추천**: 성향 진단 결과에 가장 부합하는 심리 발달 및 학습 도서 자동 추천.
- **마이페이지 성향 레포트**: 진단 기록을 아카이빙하고 시간 경과에 따른 발달 곡선을 시각화.
- **도서 좋아요 및 찜**: 마음에 드는 책을 담아두고 자녀가 읽은 책을 체크할 수 있는 책장 서비스.

## 사용 기술
- **Frontend**: React, CSS (Custom Variables)
- **Backend**: Node.js, Express, MySQL, REST API

## 내가 기여한 부분
- **성향 진단 알고리즘 구현**: 답변 가중치 기반의 유형 분석 로직 작성 (총 8개 성향 조합 도출).
- **마이페이지 대시보드 개발**: 진단 결과 요약 및 추천 도서 북마크 보관함 UI 컴포넌트 전체 구현.
- **도서 검색 및 필터링**: 성향 키워드로 도서 리스트를 조건별 조회하는 쿼리 및 프론트엔드 상태 머신 제어.

## 트러블슈팅 및 극복 과정

### 문제: 진단 결과 도서 로딩 시 지연 현상 및 UX 저하
- **현상**: 아이의 진단 결과를 판별하고 추천 도서 데이터베이스에서 성향 태그를 교차 매칭하여 프론트엔드에 렌더링하기까지 체감 응답 속도가 3초를 초과하여 진단 중 이탈율이 우려되는 상황이었습니다.
- **해결**: 결과 판별 중에는 아이들의 집중을 도울 수 있는 귀여운 '꿈틀이' 캐릭터 로딩 애니메이션과 성향 진단 요약 구문을 먼저 렌더링하고, 추천 도서 데이터는 백그라운드에서 지연 로딩(Lazy Loading) 및 메모이제이션(React.memo & MySQL 캐싱) 기법을 사용하여 전환 과정을 매끄럽게 변경하고 응답 지연에 따른 피로도를 획기적으로 개선했습니다.
